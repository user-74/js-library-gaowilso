# ClickerProgress

Optional parameters to set various options for the clicker progress bar.

Properties:
Name Type Description
height number | string

The height of the progress bar in pixels or some css amount
width number | string

The width of the progress bar in pixels or some css amount
fontSize string

The size of the percentage font, for both outer and inner percentages
fontColor string

The color of the percentage font, for both outer and inner percentages
draggable boolean

Whether the progress bar is draggable
removeWhenDone boolean

Whether to remove the progress bar when it reaches 100%
opacity number

A number between 0 and 1 for the opacity of the progress bar
gradient Array.<string>

An array of colour values for a horizontal gradient
overflow boolean

Whether the progress bar can be filled beyond 100%
image Image

An object containing an image to use as the background image for the progress bar
finishedMessage string

A message to display when the progress bar has finished loading
clickPurchases Array.<ClickerPurchase>

ClickerProgressBar specific: an array of objects of ClickerPurchase items
